3. What data do we collect?

We only collect what is strictly necessary:

Identity: first name, last name, email, password (encrypted), date of birth

Location: only when reporting a price

Activity: YALÉ points, reports, comments

Payment: only if you are a merchant (PayPal/Stripe)

Technical data: IP address (anonymized), login logs

Tools such as NGINX and Portainer are used for technical management. Logs are kept for a maximum of 7 days, accessible only to administrators.
